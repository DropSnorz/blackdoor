Bienvenue sur notre cours Android !

Préambule

Bonjour et bienvenue dans ce cours qui porte sur l'utilisation de l'API Android, dans le cadre de l'utilisation du jeu BlackDoor. Au travers de ce cours, nous allons étudier les nombreuses possibilités offertes par les classes Android, mais aussi via les classes Java.

Attention - Des bases de programmation en Java sont nécessaires pour apprécier le contenu de ce cours
Info - Pour utiliser les classes Android, il faut importer la classe java.lang.Object dans votre projet

Vous êtes prêt à vous lancer dans l'aventure ? Alors, allons-y !
